# 📊 Analyse statistique inférentielle des médias sociaux sur la santé mentale

## 📌 Description
Ce projet analyse l’impact de l’utilisation des médias sociaux sur l’équilibre de la santé mentale. L’objectif est d’identifier les relations entre le temps passé sur les réseaux sociaux et les niveaux de bien-être (faible, modéré, élevé).

## 🎯 Objectifs
- Étudier la relation entre médias sociaux et santé mentale  
- Identifier les facteurs influençant le bien-être  
- Tester des hypothèses statistiques  

## 🧪 Méthodologie
L’analyse repose sur des méthodes statistiques inférentielles :
- Test du Chi-deux  
- Test de Student  
- Test d'ANOVA 

## 📊 Données
Les données ont été nettoyées, transformées et catégorisées pour permettre une analyse fiable.

## 🛠️ Outils utilisés
- Python 
- Outils statistiques

## 📈 Résultats
Les résultats mettent en évidence des relations significatives entre le temps d’écran et le niveau de santé mentale, permettant de mieux comprendre l’impact des médias sociaux.

## 🚀 Conclusion
Ce projet permet de fournir des éléments d’aide à la décision pour un usage plus équilibré des médias sociaux.
